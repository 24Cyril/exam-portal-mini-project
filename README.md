# Exam Portal

A web-based exam portal mini project developed as part of a BTech curriculum.
The system allows students to register, take online exams, and view results,
while administrators can manage exams, questions, and evaluations.

## Features
- User authentication (Student / Admin)
- Online exams and mock tests
- Automated evaluation
- Score tracking and result display
- Secure login and session management

## Tech Stack
- Python
- Flask
- HTML, CSS, JavaScript
- SQLite (for development)
